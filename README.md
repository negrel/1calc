# :technologist: `1calc` - Simple calculator web app.

## Getting started

First, you must install dependencies:

```shell
# If you're using yarn
yarn

# If you're using NPM
npm install
```

Then you can start development server as follow:

```shell
yarn dev

# Or
npm run dev
```
